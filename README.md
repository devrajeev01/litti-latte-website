# 📬 Litti Latte Website With Contact Form with MongoDB

## 📌 Project Description
Litti Latte is a coffe website with **Contact Us** form project where users can submit their name, email, and message.  
The backend is powered by **Node.js** with **MongoDB Atlas** for storing data.  
It includes form validation, backend API integration, and smooth user feedback using success/error messages.  
This project is fully responsive and can be deployed on platforms like **Render** or **Railway**.

---

🚀 Live Project Link
The live version of the Litti Latte website can be accessed here: https://littilatte.onrender.com

- **Frontend**: HTML, TailwindCSS, JavaScript
- **Backend**: Node.js, Express.js

⚙️ Setup Instructions

1️⃣ Install Dependencies
After cloning the repository, navigate to the project directory and install the necessary dependencies by running the following command:
Bash
npm install

2️⃣ Environment Variables
Create a .env file in the root directory of your project. This file will hold your sensitive information, such as the MongoDB connection string.
PORT=3000
MONGODB_URI=<Your MongoDB Atlas connection string>

To obtain your MongoDB Atlas connection string, log in to your Atlas account, navigate to your cluster, and click Connect. Choose the "Connect your application" option and copy the provided connection string.

3️⃣ Run the Project
To start the server, use the following command:
Bash
npm start
The server will now be running at http://localhost:3000.
